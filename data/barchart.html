Brain Deaths in South Korea

Methodology: Using D3.js Graph Gallery's example, I imported brain deaths data in South Korea. I changed y axis domain into '.domain([0, 600]) so that it matches with the domain of my dataset.

But then I wanted to add tooltip on the bar. While looking for options, I found an example from this chart. I added below code along with some css.

var tooltip = d3.select("body").append("div").attr("class", "toolTip");
.on("mousemove", function(d){ tooltip
.style("left", d3.event.pageX - 50 + "px")
.style("top", d3.event.pageY - 70 + "px")
.style("display", "inline-block")
.html((d.area) + "
" + (d.value));
})
.on("mouseout", function(d){ tooltip.style("display", "none");});
});
